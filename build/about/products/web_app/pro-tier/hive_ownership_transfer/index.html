hive_ownership_transfer

f--- title: â™»ï¸ Hive ownership transferâ€‹

Hive ownership transfer allows beekeepers to transfer complete hive records, history, and data to another user account within the Gratheon platform.

Overviewâ€‹

This feature is useful when:

  • Selling hives to other beekeepers
  • Transferring management responsibilities within a commercial operation
  • Gifting or donating hives
  • Reorganizing apiaries across multiple user accounts

How it worksâ€‹

  1. Current owner initiates a transfer request from the hive management interface
  2. System generates a transfer token or invitation link
  3. Recipient accepts the transfer
  4. All hive data, including inspection history, frames, photos, and telemetry data, is transferred to the new owner
  5. Original owner loses access to the hive data (optional: can retain read-only access)

What gets transferredâ€‹

  • Hive metadata (location, name, notes)
  • Complete inspection history
  • All frame data and uploaded photos
  • AI analysis results
  • Sensor/telemetry data history
  • Treatment records
  • Feeding history

Permissionsâ€‹

  • Only the hive owner can initiate a transfer
  • Transfer can be cancelled before acceptance
  • Both parties receive email notifications
  • Transfer requires explicit acceptance from the recipient

Use casesâ€‹

Commercial operationsâ€‹

Large beekeeping operations can use this feature to:

  • Reassign hives between employees
  • Transfer hives when staff changes
  • Organize hives by geographic region or manager

Hive salesâ€‹

When selling hives, provide complete historical data to the buyer, including:

  • Past queen performance
  • Disease history
  • Productivity records
  • Environmental sensor data

Collaborative managementâ€‹

Transfer temporary management to another beekeeper when:

  • Going on vacation
  • Unable to perform inspections
  • Collaborating with mentors or consultants
